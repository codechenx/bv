#### 简介

bv 是一个在终端下查看常见生物信息学数据的查看器. bv的终端界面和键绑定代码修改自[vdtui](https://github.com/saulpw/visidata/blob/stable/visidata/vdtui.py)

 ![Screenshot](screenshots/example.png)



# 特点

- 以表格的形式在终端下查看数据
- Vim类似的键绑定
- 支持gz格式的压缩文件
